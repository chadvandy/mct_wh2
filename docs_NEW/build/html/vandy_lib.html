load_modules(path, search_override)¶

Load every file, and return the Lua module, from within the folder specified, using the pattern specified.

Parameters

  • path (str) – The path you’re checking. Local to data, so if you’re checking for any file within the script folder, use “script/” as the path.

  • search_override (str) – The file you’re checking for. I believe it requires a wildcard somewhere, “*”, but I haven’t messed with it enough. Use “*” for any file, or “.lua” for any lua file, or “/main.lua” for any file within a subsequent folder with the name main.lua.
