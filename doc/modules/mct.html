load_mod (filename, filename_for_out)
For internal use, loads specific mod files located in script/mct/settings/.
Any .lua file found in here is given the MCT manager as the variable mct within the full scope of the file.

Parameters:

  • filename string The filename being required and loaded.
  • filename_for_out string The original filename with the full directory path still included; used for outputs.
load_module (module_name, path)
Internal loader for scripts located in script/mct/modules/.

Parameters:

  • module_name string The .lua file name. Exclude the ".lua" part of it!
  • path string The path to find this .lua file. Make sure package.path has been editing before this function is called!
register_mod (mod_name)
