add_new_section (section_key, localised_name)
Add a new section to the mod's settings view, to separate them into several categories. When this function is called, it assumes all following options being defined are being assigned to this section, unless further specified with mct_option.

Parameters:

  • section_key string The unique identifier for this section.
  • localised_name string The localised text for this section. You can provide a direct string - "My Section Name" - or a loc key - "loc_key_example_my_section_name". If a loc key is provided, it will check first at runtime to see if that localised text exists. If no localised_name is provided, it will default to "No Text Assigned"
get_options_by_section (section_key)
Returns a k/v table of {option_key=option_obj} for options that are linked to this section. Shouldn't need to be used externally.

Parameters:

  • section_key string The unique identifier for this section.
